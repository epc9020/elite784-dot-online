Software

downloaders

yt-dlp - a command line interface based video downloader. can probably download videos from sites other than youtube (yet to be attempted by me)
cobalt.tools - Browser based video and audio downloader that can download videos from many sites
Free Download Manager - Free download manager. It is exactly as it sounds like. Can do normal downloads and BitTorrent. My personal favorite download manager on windows, although it is also available on Linux (isnt very stable as far as i remember, at least on fedora) and i think MacOS also.
Motrix - Download manager that can also do normal downloads and BitTorrent. My personal favorite on Linux, and i think its also available for Windows and Mac i think.

image viewers and editors

RawTherapee - A photo editor that can edit raw images.
DigiKam - Photo viewer I use to skim though my digital photographs and create panoramas with its panorama wizard (panorama feature depends on Hugin, provided below)
Hugin - Panorama creation software. DigiKam integrates this along with other software to create pano's easily, but Hugin can be utilized by itself (ive never done it)

Privacy

Adnauseam - Ad blocker based on uBlock that descreetly clicks on every blocked ad, making it so that garbage data gets sent back to advertisers, resulting in data obfuscation.
TrackMeNot - Extention that makes a random search query in the background every 10 seconds, which also results in data obfuscation.
Noscript - Extention that disables JavaScript on websites until you allow some or all scripts on that site to run

Appearance

OldTwitter - Turns "hell" back into its former "glory".
Youtube redux - Make YouTube look like when it was good. Can also do some behaiviour modifications.

Behaiviour

Old Reddit Redirect - Automatically redirects all reddit links to the old.reddit site.
Return YouTube Dislike - Returns the YouTube dislike counter
Youtube Shorts Blocker - Extention that blocks shorts on youtube. You can configure it so that YTS videos play in the full player instead of the doomscrolling feed player (this is how i have it configured)

Misc.

Translate Web Pages - Website translator. You can use this to translate entire web pages or just translate a portion of text on a site.
